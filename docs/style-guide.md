# Moki Market 设计风格速查

完整设计与 UI 规范：

- `Moki-Market-Design-UI-Spec-v1.0.pdf`

## 整体风格

- 冷静
- 研究感
- 专业
- 可信
- 中文友好

## 色彩分工

- 暖白、白色、深墨和中性灰承担主要界面。
- 哑光金承担品牌、主操作、重点数据和选中状态。
- 红棕色只承担风险、警告和免责声明。
- 绿、青、珊瑚和紫仅作为图表辅助色。

颜色必须复用 `src/app/globals.css` 中的语义变量，不新增近似金色。

## 核心原则

- 不做成交易所、券商盘口或营销落地页。
- 不使用强烈红绿涨跌刺激。
- 不大面积铺金色，不使用金色渐变和发光效果。
- 卡片只用于独立重复项和真正需要框定的工具区域。
- 主圆角为 8px，不使用大面积超大圆角。
- 研究卡适合截图传播，信息层级清楚且密度适中。
- 移动端单列优先，文字自然换行，不允许横向滚动。

## 视觉语言

- 以信息层次和阅读效率为核心。
- 强调数据来源、更新时间和置信度。
- 主按钮明显，但同一区域最多一个主 CTA。
- 风险色与品牌色严格分离。
- Hover 以边框和颜色变化为主，不使用缩放或布局位移。
