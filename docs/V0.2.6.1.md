# V0.2.6.1 - Earnings Expectations & Guidance 接入总结

## 概述
V0.2.6.1 完成了财报一致预期（Consensus Estimates）和公司指引（Guidance）的数据接入和展示逻辑。

## 已完成功能

### 1. 环境变量与配置 (Step 2)
- `EXPECTATIONS_PROVIDER` - 选择预期数据来源 (fmp | mock | manual | disabled)
- `FMP_API_KEY` - Financial Modeling Prep API Key
- `ALPHA_VANTAGE_API_KEY` - Alpha Vantage API Key
- `GUIDANCE_PROVIDER` - 选择指引数据来源 (manual | mock | disabled)
- `ENABLE_MARKET_EXPECTATION_PROXY` - 为未来市场预期预留
- `DEBUG_EARNINGS_DATA` - 启用调试 API

### 2. 类型定义 (Step 3)
- `ConsensusEstimate` - 单季度一致预期
- `ConsensusEstimateBundle` - 完整预期包
- `standardizeRevenue` / `standardizeEps` - 单位标准化

### 3. FMP Provider (Step 4)
- 从 FMP `/analyst-estimates` 和 `/earnings-surprises` 获取数据
- 完整容错处理（适配不同套餐/API版本）
- Revenue 自动转换为 billion USD
- EPS 保持 USD per share

### 4. Alpha Vantage Provider (Step 5)
- 从 Alpha Vantage `EARNINGS_ESTIMATES` 获取数据
- 支持不同格式的解析
- Rate limit 检测与静默降级

### 5. Provider 统一调度 (Step 6)
- `getConsensusEstimates(ticker)` - 统一入口
- 根据配置选择主 provider
- 无结果时自动 fallback
- 不使用 mock 填充真实页面

### 6. 季度匹配逻辑 (Step 7)
- `selectCurrentQuarterConsensus` - 选择当前季度预期
- `selectNextQuarterRevenueConsensus` - 选择下季度营收预期
- 优先级：精确季度匹配 → 日期接近 → 最新季度（best-effort）
- 完整 debug reason 输出

### 7. 接入 earnings snapshot (Step 8)
- 优先使用 consensus estimate
- fallback 到原 Yahoo/FMP 数据
- 保留 actuals 优先 SEC/Eastmoney/Yahoo
- surprise 自动计算
- 不改变 UI props

### 8. 手动 Guidance Provider (Step 9)
- 人工维护的 core tickers 指引
- 支持 NVDA/ORCL/SNOW/DELL
- 与 SEC evidence 结合
- V0.2.7 将替换为自动解析

### 9. Guidance + Consensus 整合 (Step 10)
- 计算 guidance surprise pct
- 整合 manual guidance 与 consensus
- GuidanceComparePanel 展示完整信息
- 无数据时温和空状态

### 10. SEC Guidance Parser 占位 (Step 11)
- `searchTextForGuidanceEvidence` - 关键词搜索（outlook/guidance/expected to be等）
- `parseGuidanceFromText` - 占位解析函数
- 从现有 SEC submissions 获取 filing info
- 不强制解析成数值，先展示证据

### 11. Debug API (Step 12)
- `GET /api/debug/earnings-data?query=NVDA`
- 完整调试信息：
  - Resolved ticker
  - Config summary
  - Actuals status
  - Consensus provider status
  - Selected estimates with reasons
  - Guidance status
- Production 默认禁用

### 12. 前端文案更新 (Step 13)
- FMP → "Consensus 来源：FMP analyst estimates。"
- Alpha Vantage → "Consensus 来源：Alpha Vantage earnings estimates。"
- Best-effort → "当前预期数据为 best-effort 匹配，后续将接入财报前快照数据库。"
- 无 consensus → "暂未获取到该公司的市场一致预期。"

### 13. Market Expectation Proxy 预留 (Step 14)
- `impliedMovePct` - 期权隐含波动
- `preEarningsPriceMove7dPct` / `30dPct` - 财报前价格走势
- `crowdRevenueEstimate` / `crowdEpsEstimate` - 社群预期
- 预留 V0.2.7/V0.3 接入 Estimize/ORATS/Intrinio

## 配置示例

```env
# .env.local
SEC_USER_AGENT="Moki Market contact@example.com"
FMP_API_KEY="your_fmp_key"
ALPHA_VANTAGE_API_KEY="your_av_key"
EXPECTATIONS_PROVIDER=fmp
GUIDANCE_PROVIDER=manual
ENABLE_MARKET_EXPECTATION_PROXY=false
DEBUG_EARNINGS_DATA=true
```
