# v0.5.2 Technical Data Adapter Real

v0.5.2 turns the Technical Structure Dashboard from a visible mock surface into a live adapter-backed research module when Yahoo chart data is available. It keeps the v0.5.1 fallback path so the report remains usable when market data cannot be fetched.

v0.5.2-hotfix adds the missing visual chart surface: the dashboard now renders a candlestick K-line chart with EMA overlays and volume bars instead of only showing calculated indicator cards.

## Implemented

- Advanced `ResearchReport.schemaVersion`, route cache keys, and ResearchReport prompt cache key to `v0.5.2`.
- Added `TechnicalDataSnapshot` as an optional first-class field on `ResearchCard` and `ResearchReport`.
- Added server-side Yahoo chart daily K-line fetcher at `src/lib/globalStockData/yahooChart.ts`.
- Added market technical adapter at `src/lib/research-report/marketTechnicalDataAdapter.ts`.
- Added `/api/technical-data` with GET `?query=` and POST `{ card }` support.
- Calculated MA20/MA50 trend context, RSI14, 20d realized volatility, 20d volume pressure, 60d benchmark relative strength, 20d support/resistance, and 6mo range.
- Updated `/generate` to request technical data in parallel and merge the snapshot into ResearchReport, TechnicalDashboard, and IntegratedReport readiness.
- Updated the Technical Dashboard UI to distinguish live Yahoo K-line adapter output from legacy fallback output.
- Added `lightweight-charts` for candlestick chart rendering.
- Added `TechnicalDataSnapshot.chart` with OHLCV bars, EMA5/10/20/50, range, interval, currency, and data-as-of metadata.
- Added `TechnicalKLineChartPanel` to render daily K-line candlesticks, EMA overlays, volume bars, quote summary, and latest OHLC values.

## PDF Planning Match

- Matches the planned Yahoo chart / global-stock-data K-line direction by using the existing global-stock-data symbol mapping layer and Yahoo chart endpoint.
- Preserves the ResearchReport-centered architecture: K-line data becomes `TechnicalDataSnapshot`, then powers dashboard and integrated readiness.
- Completes the missing UI surface from the plan by showing the K-line chart itself, not only derived technical indicators.
- Keeps investment-product boundaries intact: no target price, rating, trading instruction, or recommendation is generated.
- Keeps graceful fallback: if Yahoo chart or benchmark data fails, dashboard status becomes partial/fallback and warnings are surfaced.

## Still Not Included

- No MACD, Bollinger Bands, KDJ, full intraday charting, or full technical trading system.
- Period tabs beyond daily K-line are not wired yet.
- No persistent market-data database.
- No automatic provider rotation beyond Yahoo chart.
- No source discovery or filing/transcript web crawling.
