# v0.5.1 Technical Structure Dashboard Mock

## Goal

v0.5.1 adds a visible Technical Structure Dashboard product surface to `/generate`. It is intentionally a mock built from the current `ResearchReport`, buy-side report monitoring plan, and legacy technical context. It does not connect Yahoo chart, K-line data, live technical indicators, or benchmark relative strength.

## Completed

- Advanced the runtime report schema marker to `v0.5.1`.
- Advanced route cache and ResearchReport prompt cache keys to `v0.5.1`.
- Updated the technical dashboard builder semantics from adapter-first to `mock_from_research_report`.
- Renamed the generated dashboard title to `Technical Structure Dashboard`.
- Updated technical dashboard warnings and disclaimer to state that live K-line and calculated indicators are unavailable in v0.5.1.
- Added a `Technical Structure Dashboard Mock` section to the `/generate` generated report output.
- Reused `TechnicalDashboardPanel` to expose indicator matrix, key zones, scenario read-through, and technical readiness.
- Added visible live-data and gap badges so the mock cannot be mistaken for a connected market-data dashboard.
- Updated `/generate` hero capability labels to include `Technical Structure`.

## PDF Alignment

- Matches the PDF direction that Technical Structure Dashboard should become a visible product section, not a hidden schema artifact.
- Presents technical state as a structured dashboard with indicators, zones, scenario read-through, and readiness.
- Keeps the boundary explicit: this is a mock from ResearchReport context, not live market data.
- Prepares the UI surface for the next version to connect a real technical data adapter.

## Still Not Done

- No Yahoo chart or global-stock-data K-line data.
- No moving averages, RSI, MACD, volatility bands, volume profile, or benchmark relative strength calculations.
- No interactive chart canvas.
- No live market data refresh or technical provider selection.

## Next Version

v0.5.2 should connect a real Technical Data Adapter:

- Pull K-line/quote data through Yahoo chart or global-stock-data compatible sources.
- Normalize OHLCV into a technical snapshot.
- Compute basic indicators and benchmark relative strength.
- Feed the real technical snapshot into the existing Technical Structure Dashboard surface.
