# v0.4.5 Technical Data Adapter

## 目标

v0.4.5 在 v0.4.4 Technical Dashboard Mock 之后新增 Technical Data Adapter。当前版本先建立本地 adapter contract，把 legacy `technicalContext` 和 `technical_context` section 规范化为 `TechnicalDataSnapshot`，再由 snapshot 驱动 dashboard。它不联网抓取实时行情，也不计算技术指标。

## 已完成

- 将 `ResearchReport.schemaVersion` 更新为 `v0.4.5`。
- 新增 `TechnicalDataSnapshot`、`TechnicalDataPoint`、`TechnicalDataZone`、`TechnicalDataProvider` 和 `TechnicalDataAdapterStatus` 类型。
- 扩展 `ResearchReport.technicalDashboard.summary`，新增 `adapterStatus`、`provider`、`liveDataAvailable` 和 `dataAsOf`。
- 新增 `src/lib/research-report/technicalDataAdapter.ts`，把 legacy 技术文案和关键区间适配为结构化技术数据快照。
- `buildResearchReportFromCard` 改为调用 `buildTechnicalDashboardFromAdapter`。
- `TechnicalDashboardPanel` 现在展示 Adapter ready、provider、live data 状态和每个指标的 data status。
- `technicalDashboardMock.ts` 保留为兼容 wrapper，避免历史路径断裂。

## Adapter 行为

- 当前 provider 固定为 `legacy_technical_context`。
- `adapterReady = true` 表示 legacy 技术内容已被规范化，不表示实时行情已接入。
- `liveDataAvailable = false` 明确说明 v0.4.5 还没有价格、成交量、期权链或 K 线 provider。
- 没有 evidence/fact/source 绑定的技术项继续显示 `requires_review` 或 `needs_source`。
- 不生成买卖点、止损位、仓位建议、技术评分或交易信号。

## 当前未做

- 不接入实时行情 provider。
- 不计算均线、RSI、MACD、ATR、成交量倍率、IV percentile 或支撑/压力。
- 不调用外部 API，不联网抓取历史价格。
- 不迁移 `/generate` 动态链路。
- 不进入 v0.4.6 integrated report 的最终整合展示。

## 后续交接

v0.4.6 Integrated Research Report 可以读取：

- `ResearchReport.technicalDashboard.dataSnapshot`
- `ResearchReport.technicalDashboard.summary`
- `ResearchReport.technicalDashboard.indicators`
- `ResearchReport.technicalDashboard.zones`
- `ResearchReport.buySideReport`
- `ResearchReport.evidenceLayer`

如果后续接入真实技术数据 provider，应替换 `TechnicalDataSnapshot.provider`、`points`、`zones` 和 `liveDataAvailable`，但继续保留 review status、source ids、warnings 和 missing references。
